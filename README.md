# React Projects Collection

This repository contains various React.js projects showcasing different concepts, features, and best practices in modern React development.

## Projects

### 1. Product List Application (My_App)

A modern React application that displays a product catalog using the Fake Store API.

#### Key Features
- Responsive product grid layout
- Custom data fetching hook (useFetch)
- Loading state management
- Error handling
- Tailwind CSS styling
- Modern UI with hover effects

#### Tech Stack
- React 19.1.1
- Vite 7.1.7
- Tailwind CSS 4.1.16
- ESLint

[View Product List App Details](./My_App/README.md)
